# 기능 목록
## 도메인 로직
### 자동차 이름 유효성 검사
[] 통과 조건
- 영어,한글만 가능(띄어쓰기 불가)
- 글자 수: 1~5자
- 쉼표로 이름들 구분 (쉼표가 존재, 쉼표 개수 = 경주 참여 이름 개수 -1)

### 진행 횟수 유효성 검사
[ ] 통과 조건
- 정규식 표현 사용(정수만 인식)
- 진행 횟수: 1~5회

### 랜덤 숫자 생성
- 0에서 9 사이의 정수에서 무작위 값을 구하기

### 자동차
- 필드 속성: 이름, 전진 횟수
- 생성자 파라미터: 이름 

#### 전진 판단
- 랜덤 숫자(정수)가 4이상 일 경우 전진(전진 횟수 +1)

## Input, OutPut
[] Input
- 사용자의 입력값 받아오기

[] Output
- 메세지 출력하기
